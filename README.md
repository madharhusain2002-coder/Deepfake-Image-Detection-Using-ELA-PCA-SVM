# Deepfake Image Detection Using ELA, PCA and SVM

## Overview
This project presents a classical forensic approach for detecting deepfake images using Error Level Analysis (ELA), Principal Component Analysis (PCA), and Support Vector Machine (SVM).

## Features
- Error Level Analysis (ELA)
- Feature Extraction
- Principal Component Analysis (PCA)
- Support Vector Machine (SVM)
- Deepfake Image Classification

## Technologies Used
- Python
- OpenCV
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Pillow
